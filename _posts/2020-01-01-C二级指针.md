# 二级指针
- 二级指针保存的是一级指针的地址，三级、四级同理
- 如果指针做实参，那么传实参的指针（地址），就会用到二级指针
- 要改变指针的值就需要用到指针的值，即二级指针
- 二级指针作用：传参时改变一级指针的值

void f(int **ptr1, int *ptr2)
{
	int *tmp = ptr2;
	**ptr1 *= 10;
	*ptr2 *= 10;
	ptr2 = *ptr1;
	*ptr1 = tmp;
}
