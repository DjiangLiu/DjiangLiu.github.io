# 概述
直接对整数在内存中的二进制位进行操作。效率更高
1. 与 and &
2. 或 or |
3. 取反 not ~
4. 异或 xor ^
5. 左移 shl <<
6. 右移 shr >>
7. 逻辑与 && 逻辑或|| 逻辑取反！ 

# 与运算
二进制数计算，都为1.则为1。

```c
    1111
&   1001
--------
    1001
```
## 作用
1. 获取其中几位。
2. IP & 子网掩码
3. 获取一个数字中有多少个1。整数a二进制中最右边的1

    a&(a-1)

# 或运算
二进制数计算，都为0，则为0。

```c
    1111
|   1001
--------
    1111
```

# 异或运算
1. 二进制数计算，相同为0，不同为1。任何数和0异或等于自身，与自身异或等于0。
2. 可以在不添加临时变量的情况下交换两个数


```c
    1111
^   1001
--------
    0110
```

```c
a = a^b;
b = a^b;
a = a^b;

a = a+b;
b = a-b;
b = a-b;
```
