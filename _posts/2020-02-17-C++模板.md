# 冒泡排序
1. i < nLen - 1 - k 每次循环k最后一个值都是确定的，所以可是少循环一次


```c
#include<stdio.h>
#include<math.h>

void Sort(int* arr, int nLen)
{
	for (int k = 0; k < nLen-1; k++)
	{
		for (int i = 0; i < nLen - 1 - k; i++)
		{
			if (arr[i] > arr[i + 1])
			{
				int temp = arr[i];
				arr[i] = arr[i + 1];
				arr[i + 1] = temp;
			}
		}
	}
}

int main()
{
	int arr[] = { 111,222,333,4,5,6,7,8,9,11,22 };
	Sort(arr, 11);
	for (int j = 0; j < 11; j++)
	{
		printf("%d\n", arr[j]);
	}
	return 0;
}
```
