# 定义
1. 一个函数直接或间接的调用自身。
2. 首先要有一个或多个递归出口，即递归的终止条件，也就是最小问题的求解。
3. 递归还需要一个递归式，规定了如何将原问题划分为子问题（子问题解决的对象和原问题解决的对象性质一样）。

## 阶乘

```c
#include<stdio.h>
#include<math.h>

int Test(unsigned int n)
{
	if (n == 0 || n == 1)
	{
		return 1;
	}
	else
	{
		return n* Test(n - 1);
	}
}

int main()
{
	int a = Test(4);
	printf("%d\n", a);
	return 0;
}
```

## 斐波那契额数列

```c
#include<stdio.h>
#include<math.h>

int feibo(unsigned int n)
{
	if (n == 1 || n == 2)
	{
		return 1;
	}
	else
	{
		return feibo(n - 1) + feibo(n - 2);
	}
}

int main()
{
	int a = feibo(4);
	printf("%d\n", a);
	return 0;
}
```
