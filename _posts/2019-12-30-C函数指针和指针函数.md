# 函数指针
指向函数的指针，如果一个指针变量存放的是函数的地址，那么就把这个指针成为函数指针。
## 函数的签名
包含函数的返回值类型、参数类型、参数个数，函数指针的类型由对应的函数的签名决定
## 定义函数指针的方式
1. 第一种用typedef定义出函数的类型，然后通过指针类型来定义函数
2. 第二种用函数签名来决定函数的指针

### 应用
1. hook
2. dll函数动态调用
3. 加壳

# 指针函数
返回值是指针的函数

## 这段代码的问题在何处
```c
//修正前
#include<stdio.h>
#include<malloc.h>
#include<string.h>

void getmem(char* p)
{
    //3.这里分配完100字节并未释放空间，内存泄露
	p = (char*)malloc(100);
}

int main(void)
{
	char* str = NULL;
	char s[] = "hello";
	//1.这里是传值，所以不会改变实参
    getmem(str); 
    //2.strcpy存在溢出的情况
	strcpy(str, s);
	printf(str);
	return 0;
}
```
```c
//修正后1：利用指针函数
#include<stdio.h>
#include<malloc.h>
#include<string.h>

char* getmem()
{
	char* p = (char*)malloc(100);
	return p;
}

int main(void)
{
	char* str = NULL;
	char s[] = "hello";
	str = getmem();
	strcpy_s(str, 100, s);
	printf(str);
	free(str);
	str = NULL;
	return 0;
}
```
```c
//修正后2：利用传址的方式
#include<stdio.h>
#include<malloc.h>
#include<string.h>

void getmem(char**p)
{
	* p = (char*)malloc(100);
}

int main(void)
{
	char* str = NULL;
	char s[] = "hello";
	getmem(&str);
	strcpy_s(str, 100, s);
	printf(str);
	free(str);
	str = NULL;
	return 0;
}
```
```c
//修正后3：传引用
#include<stdio.h>
#include<malloc.h>
#include<string.h>

void getmem(char *& p)
{
	//先取p的内存地址，然后在定义指针
	p = (char*)malloc(100);
}

int main(void)
{
	char* str = NULL;
	char s[] = "hello";
	getmem(str);
	strcpy_s(str, 100, s);
	printf(str);
	free(str);
	str = NULL;
	return 0;
}
```

# 不能返回局部变量的指针
局部变量的生存周期只在函数内部，在函数结束的时候，指针就结束了。


# 作业
1. `int add(int x,int y)`函数指针
```c
#include<stdio.h>

int add(int x, int y)
{
	return x + y;
}

typedef int(*F)(int x,int y);//函数指针类型

int main(void)
{
	int a = 10;
	int b = 20;

	F f1 = &add;
	//这里有没有&都一样的
	F f1 = add;
	int z = f1(a, b);

	int(*f2)(int x, int y) = add;
	int x = f2(a, b);

	int c = add(a, b);
	printf("z:%d\nx:%d\nc:%d\n", z, x, c);
	getchar();
	return 0;
}
```
